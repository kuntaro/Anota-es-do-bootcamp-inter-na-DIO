# Logíca de programação essencial



- **Metacognição**
  - A metacognição é ter consciência de seus atos e pensamentos.
  - *Exemplo:*
    - "início - dia: acordei - levantei da cama - troquei de roupa - escovei os dentes - fui a padaria - etc..."
- **Pseudocódigo** 
  - O pseudocódigo é uma forma genérica de escrever um algoritmo, utilizando uma linguagem simples.
  - Com um pseudocódigo e usando um algoritmo você consegue concluir uma atividade.
- **Fluxograma**
  - Fluxograma é uma ferramenta utilizada para representar graficamente o algoritmo, isto é, a sequência lógica e coerente do fluxo de dados um fluxograma é um tipo de diagrama e pode ser entendido como uma representação esquematicamente de um processo podemos entender na prática como a documentação dos passos necessários para a execução de um processo qualquer.
- **Variáveis**
  - É um objeto (uma posição, frequentemente localizada na memoria  capaz de reter e representar um valor ou expressão).
- **Constante**
  - São valores imutáveis e não são alterados durante a vida útil do programa.
- **Concatenação**
  - É um termo usado em computação para designar a operação de unir o conteúdo de duas **strings** (string é uma sequência de caracteres). É o agrupamento de duas ou mais células que incluindo formulas, textos ou outras informações contidas no seu interior, da origem a um único resultado.

- **Estrutura de repetição**

  - Dentro da lógica de programação é uma estrutura que permite executar mais de uma vez o mesmo conjunto de comandos, de acordo com uma condição ou um contador.

- **Linguagem de programação**

  - é uma linguagem escrita formal que específica um conjunto de instruções e regras usadas para gerar programas (software), u, software pode ser desenvolvido para rodar em um computador, dispositivo móvel ou em qualquer equipamento que permita sua execução.

- **Matriz**

  - É uma coleção de variáveis dos mesmos tipos acessivos com um único nome e armazenados contiguamente na memória. A individualização de cada variável de um vetor é feita através do uso de índices. Os vetores são matrizes de uma só dimensão.

- **O que é estrutura de dados**

  - É uma estrutura organizada de dados na memória de um computador ou em qualquer dispositivo de armazenamento de forma que os dados possam ser utilizados e forma correta.

- **Algoritmo** 

  - Um algoritmo é um conjunto de instruções básicas estruturadas e ordenadas, seu objetivo é realizar uma tarefa ou operação específica (algoritmo é o programa em si).

    Em uma estrutura de dados devemos saber como realizar um determinado conjunto de operações básicas, como:

    - *Inserir dados*;

    - *Excluir dados;*

    - *Localizar um elemento;*

    - *Percorrer todos os itens constituintes da estrutura para visualização;*

    - *Classificar, que se resume em colocar os itens de dados em uma determinada ordem.*
